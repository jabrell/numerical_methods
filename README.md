# Numerical Methods and Optimization

The repo provides additional material for the course *Numerical Methods and Optimization*
at the University of Basel. 

## Notebooks

We use jupyter notebooks for Python codes. You can either run the notebooks locally
(see description below) or run the notebooks in Google Colab (requires Google account).
To run a notebook in Colab, in github go to the notebook (identified by the ending .ipynb)
and use the button at the top of the notebook.


## Local installation
To use the material locally, use [uv](https://docs.astral.sh/uv/getting-started/installation/).
First install it. Then clone or download the folder:

```bash
git clone https://github.com/jabrell/numerical_methods.git
cd numerical_methods
```

In the main folder, i.e., the one that contains this *README.md* file run
```bash
uv sync
```
